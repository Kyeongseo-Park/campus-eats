3. 공통 컴포넌트 명세 (Component Specification)

3.1 Global Navigation Bar (GNB / Bottom Nav)

아이템: 홈, 검색, 찜, 마이

스타일: 오렌지 컬러(primary) 강조, 활성화 상태 아이콘 및 라벨 변경

특징: 화면 하단 고정, 부드러운 전환 효과

3.2 Top App Bar

헤더: 브랜드 로고(학식말고 뭐먹지?) 또는 화면 타이틀

기능 아이콘: 알림(Notifications), 메뉴(Menu) 또는 뒤로가기(Back) 버튼

3.3 Filter Chips (Quick Filters)

구역: 정문, 상대, 예대, 후문, 공대쪽문

카테고리: 한식, 중식, 일식, 양식, 분식, 카페

상태: 선택 시 배경색 오렌지로 반전, 텍스트 굵게 처리

3.4 Restaurant Card (List Item)

구성 요소: 대표 이미지, 식당명, 평점(별점), 카테고리 뱃지, 위치 요약 정보, 찜 버튼(하트)

동작: 카드 클릭 시 식당 상세 페이지 이동

3.5 Bottom Sheet / Modal

식당 정보 시트: 지도 마커 클릭 시 하단에서 슬라이드 업

상세 필터 모달: 검색 탭에서 필터 버튼 클릭 시 전체 화면 덮는 형태



4. 디자인 시스템 (Design System)

주요 컬러: #FF6B00 (Main Orange)

폰트: Plus Jakarta Sans

라운드: Round 8 (8px) 적용으로 부드러운 UI 지향

톤앤매너: 밝고 활기찬 대학생 타겟의 캐주얼한 무드



5. 핵심 비즈니스 로직

위치 권한: 거부 시 '정문' 기준으로 거리 계산 안내 팝업 노출

로그인 분기: 리뷰 작성, 식당 제보 시 비로그인 유저에게 로그인 유도 모달 노출

제보 프로세스: 제보 등록 -> 관리자 검토 -> 마이페이지 '내 제보 확인'에서 상태(대기/승인/반려) 업데이트 확인
